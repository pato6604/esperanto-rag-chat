Esperanto Architecture

RAG documental con aislamiento por sesión · Hybrid Search · FastAPI + Next.js + Qdrant + Gemini

Stack

  • • Next.js 16 + React + TypeScript
  • • FastAPI + Python 3.14
  • • Qdrant (vector DB, local mode)
  • • Gemini API (embeddings + chat)
  • • Tailwind CSS + shadcn/ui

Features

  • • Session Isolation (per-session Qdrant filter)
  • • Hybrid Search (Vector + BM25 + RRF fusion)
  • • Cross-Session Detection
  • • SSE Streaming
  • • Multi-format ingestion (PDF, DOCX, CSV, JSON, HTML)

Deployment

  • • Local dev: hot reload (uvicorn + next dev)
  • • Railway-ready backend (FastAPI)
  • • Vercel-ready frontend (Next.js)
  • • Qdrant Cloud-ready (one config toggle)
